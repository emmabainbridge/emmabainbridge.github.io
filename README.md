# Emma Bainbridge — Minimal Academic Site (GitHub Pages)

This is a custom, minimalist site designed to feel like a boutique landing page rather than a stock academic template. It’s pure HTML/CSS/JS (no theme), so it’s fast and easy to tweak.

## Quick start

1. Create a new public repo named `<your-username>.github.io`.
2. Upload the contents of this folder to the repo root and commit.
3. Visit `https://<your-username>.github.io` after a minute or two.

### Custom domain (optional)
- In your repo, go to **Settings → Pages** and add your domain under **Custom domain**. 
- At your DNS provider, point the apex A records to GitHub Pages IPs and `www` to `<username>.github.io` via CNAME.

## Customize
- Edit `index.html` text and links.
- Replace `assets/img/og-card.png` and `assets/img/favicon.svg`.
- Add your CV to `files/Emma_Bainbridge_CV.pdf` (create the `files` folder).

## Notes
- The layout uses a sticky header, large hero, and compact sections.
- CSS variables live in `assets/css/style.css`.
